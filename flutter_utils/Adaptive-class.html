Adaptive

Adaptive class abstract

Abstract class that provides an adaptive layout system for Flutter widgets.

This class defines the interface and default behavior for building different widget trees based on the device's size. It provides methods to determine whether to use the mobile or desktop layout and animates the transition between these layouts.
